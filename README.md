# Stratpoint Engineering Hub

The central knowledge base for Stratpoint's engineering teams — guidelines, golden paths, best practices, and onboarding guides across all engineering domains.

## Domains

- **Web** — Frontend standards, golden paths, code review checklists
- **Mobile** — Flutter and native iOS/Android guidelines
- **Data** — Data engineering best practices and pipelines
- **Cloud** — Infrastructure, cloud architecture, and deployment
- **AI/ML** — AI-assisted development, SDLC framework, model integration

## Tech Stack

- [Nextra 4](https://nextra.site) — Documentation framework
- [Next.js 16](https://nextjs.org) — App Router
- [pnpm](https://pnpm.io) — Package manager

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Contributing

Each domain has its own section under `src/content/`. Add or update `.mdx` files following the existing structure. All changes go through a PR — no direct pushes to `main`.

# Mobile Engineering

The Mobile domain covers all native and cross-platform mobile application development at Stratpoint. We support two primary stacks: React Native (Expo) for cross-platform projects and Flutter for projects where it is the client's preference.

## Key Technologies

| Category | React Native Stack | Flutter Stack |
|----------|--------------------|--------------|
| Framework | React Native (Expo) | Flutter |
| Language | TypeScript | Dart |
| Navigation | Expo Router | GoRouter |
| State Management | Zustand / TanStack Query | Riverpod / Bloc |
| Testing | Jest, Detox | Flutter Test, integration_test |
| Build & Distribution | EAS Build | Fastlane, Codemagic |

# Backend Engineering

The Backend domain covers all server-side application development at Stratpoint. This includes REST and gRPC microservices, event-driven systems, and API platforms built across multiple languages and frameworks.

## Key Technologies

| Category | Primary Stack |
|----------|--------------|
| Frameworks | NestJS, Fastify, Spring Boot, Django, FastAPI |
| Languages | TypeScript, Java, Python |
| Databases | PostgreSQL, MySQL, MongoDB, Redis |
| Messaging | Kafka, RabbitMQ, AWS SQS |
| API Standards | REST, gRPC, GraphQL |
| Testing | Jest, JUnit, pytest |
| Deployment | Docker, Kubernetes, AWS ECS |

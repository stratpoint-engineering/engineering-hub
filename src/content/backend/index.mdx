# Backend

Welcome to the Backend capability domain. This section covers engineering standards, golden paths, and best practices for building backend services at Stratpoint.

import { Callout, Cards, Card } from 'nextra/components'

## What's Here

- **Golden Paths** — Opinionated starter configurations for common backend stacks used at Stratpoint, including NestJS, Fastify, Spring Boot, and Django.

## Domains We Cover

- REST and gRPC microservices
- Java / Spring Boot services
- Python / Django and FastAPI services
- Node.js / TypeScript services (NestJS, Fastify)

## Contributing

If you'd like to contribute a golden path, best practice, or onboarding guide for a backend stack you use on a project, open a pull request against the Engineering Hub repository.

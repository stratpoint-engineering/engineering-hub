# Cloud Engineering

The Cloud domain covers cloud infrastructure design, Infrastructure-as-Code, platform engineering, and site reliability at Stratpoint.

## Key Technologies

| Category | Primary Stack |
|----------|--------------|
| Cloud Providers | AWS (primary), GCP |
| IaC | Terraform, AWS CDK |
| Containers | Docker, Amazon ECS, Kubernetes (EKS) |
| CI/CD | GitHub Actions, AWS CodePipeline |
| Observability | CloudWatch, Datadog, Grafana |
| Secrets | AWS Secrets Manager, Parameter Store |
| Networking | VPC design, ALB, CloudFront, Route 53 |

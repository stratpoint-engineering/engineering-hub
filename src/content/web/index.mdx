# Web Engineering

The Web domain covers all frontend and full-stack web application development at Stratpoint. This section documents our standards, golden paths, and practices for building reliable, performant, and maintainable web products.

## Key Technologies

| Category | Primary Stack |
|----------|--------------|
| Framework | Next.js (App Router), React 18+ |
| Language | TypeScript |
| Styling | Tailwind CSS |
| State Management | Zustand / TanStack Query |
| Testing | Vitest, React Testing Library, Playwright |
| Build / Tooling | Turbopack, ESLint, Prettier |
| Deployment | Vercel, AWS (via CDK) |
